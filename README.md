# 1d1java

---

1 day 1 coding with Java

## Requirement

* [Java](https://www.java.com) (v8)
* [Gradle](https://gradle.org) (v6.8)

## Build all

```bash
$ ./gradlew build
```

## Test

```bash
$ ./gradlew test
```

## Lint Check

```bash
$ ./gradlew lintCheck
```

## Lint Format

```bash
$ ./gradlew lintFormat
```
