## Requirement

* [Java](https://www.java.com) (v8)
* [Gradle](https://gradle.org) (v9.0.0)

## Build all

```bash
$ ./gradlew build
```

## Test

```bash
$ ./gradlew test
```

## Lint Check

```bash
$ ./gradlew lintCheck
```

## Lint Format

```bash
$ ./gradlew lintFormat
```
